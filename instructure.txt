 ИНСТРУКЦИЯ ДЛЯ ИМПОРТА БАЗЫ ДАННЫХ 
(В приложении XAMPP запустить Apache и MySQL, нажать на кнопку admin в строке MySQL)
На странице phpMyAdmin cоздать базу данных под названием hogwarts
Зайти в эту созданную базу данных
На веръней панели стоит кнопка Импорт (Import), нажать на нее
Выбрать файл для импорта (hogwarts.sql обычный файл или hogwarts.sql в виде zip архива)
Нажать на кнопку вперед в правом нижнем углу
Готово

----------------------------------------------------------------------------------------
Функционал: есть 3 запускаемых класса: Server, Admin, StudentConnectionbetween;
StudentConnectionbetween это класс клиента;

Admin это класс админа, можно запустить сам класс.При запуске класса откроется окно админа.
В классе Admin вы можете добавлять книги, посмотреть какие книги есть и удалять книги по их id.

StudentConnectionbetween класс студента. При запуске вы увидите меню с авторизацией если студент
зарегистрирован то вы зайдете по login and password. Если нет то нажав на Signup вы можете 
зарегистрироваться самостоятельно.

-------------------------------------------------------------------------------------------

2) Проверьте подключен ли MySQL connector 8.0.15;
3) Запустите приложение XAMPP и запустите Apache и MySQL;

5) Запустите класс StudentConnectionbetween (Если вы хотите запустить за студента)
